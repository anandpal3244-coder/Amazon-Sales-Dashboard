## ✨ Project Overview

This project presents an **interactive Amazon Sales Dashboard** designed to transform raw sales data into actionable business insights.

The dashboard helps stakeholders understand:

✔ Sales Performance
✔ Customer Behavior
✔ Regional Trends
✔ Product Category Insights
✔ KPI Monitoring

Built using **Power BI**, **Python**, and **Data Analytics techniques**.
